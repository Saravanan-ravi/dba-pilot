We, The IT-Cell of CCM/PM Office, developers of this application for the internal use for PRS Database Administration of CCM/PM/Office, Southern Railway.

     This application is developed in Angular (Version: 17.1.3 on the initial release), Node (Version: 20.11.0), MongoDB Compass (Version: 1.42.2), Nodemon (Version: 3.0.3 for underdevelopment build). Application Production Mode launched using 'PM2' Server Command on 'Windows Server 2019 Edition'.

     This is a Stable 'Mean Stack Application' of Version 1.1.5 developed in 'Visual Studio Code' IDE with the following features:

  • User Registration, Login and Forgot Password Feature.
  • Passwords are 'End-to-End Encryted' even developers cannot see.
  • Token-based login and logout system.
  • User-Idle Timeout, which logs out the user if inactive for 15 Minutes.
  • PDF Uploader.
  • Smooth UI/UX using Font-Awesome Icons and Google fonts for a better user experience and user-friendly.
  • Quick access to Important Railway Sites.
  • Ease follow through of Job/Tasks. Prevents hiccups in the work-flow.
  • It is a go-green project.
  • No additional expenses were spent for the development of this application. Library, dependencies and resources are used from Open Source by Google and Angular.
